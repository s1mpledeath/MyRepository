Zer0

(рус. версия Зер0 )

Игровой персонаж класса Ассасин (англ. Assassin)
Его происхождение и настоящее имя на данный момент неизвестны. Вероятнее всего, персонаж получил свое прозвище, "Зер0", из-за нуля, который высвечивается на его лицевой панели после убийства противника. Также там высвечиваются и другие символы (например :D), их появление связано с выполнением различных действий. Большинство своих фраз выражает с помощью хайку.

Maya

(рус. версия Майя )

Персонаж класса Сирена (англ. Siren).
Одна из сирен – женщин, обладающих сверхъестественными способностями. На левом бедре носит книгу – модификатор класса.

Salvador

(рус. версия Сальвадор )

Это персонаж класса Шизострел (Gunzerker).
Основной отличительной способностью Сальвадора является владение приемами Шизострельбы, причём с использованием любого оружия
Само название его класса состоит из сочетания слов "оружие" и "берсеркер"
